# Releaser
> Its is a simple tool to help deploy NodeJS projects that uses **gitflow**

![](print.png)

## Installation

OS X & Linux:

```sh
yarn
```

## Usage example

node index **absolute path of git repository]**

### Example

node index **/$USER/project-path**
