# Releaser
> Its is a simple tool to help deploy NodeJS projects that uses **gitflow**

![](header.png)

## Installation

OS X & Linux:

```sh
yarn
```

## Usage example

node index.js **absolute path of git repository]**

### Example

node index.js **/$USER/project-path**
